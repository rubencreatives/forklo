Privacy Policy

Last updated: March 5, 2026

Overview

Forklo is a free Chrome browser extension that extracts recipes from food blogs and displays them in a clean, readable format. We are committed to protecting your privacy. Forklo does not collect, transmit, or share any personal data.

Data collection

Forklo does not collect any data. Specifically:

  • No personal information is collected (name, email, etc.)
  • No browsing history is tracked or recorded
  • No analytics, telemetry, or usage data is gathered
  • No data is sent to any external server
  • No cookies are set by the extension
  • No third-party services or SDKs are integrated

Data storage

When you save a recipe, it is stored locally in your browser using Chrome's built-in storage API (chrome.storage.local). This data never leaves your device and is only accessible to the Forklo extension. If you uninstall Forklo, all saved data is automatically removed.

Permissions

Forklo requests the following browser permissions, each with a specific purpose:

  • activeTab: Allows Forklo to read the current page to detect and extract recipe content when you activate the extension.
  • scripting: Required to inject the recipe parser and side panel into the active page.
  • storage: Used to save your recipe collection locally in the browser.
  • sidePanel: Opens the Forklo panel in Chrome's built-in side panel area.

Single purpose

Forklo has a single, clearly defined purpose: to extract and display recipes from web pages in a clean, readable format. All features — recipe parsing, ingredient checkboxes, step tracking, recipe saving, and ingredient substitution suggestions — directly support this core purpose.

Remote code

Forklo does not execute any remote code. All functionality is contained within the extension package. No external scripts, APIs, or services are loaded at runtime.

Third parties

Forklo does not share data with any third parties. There are no advertising networks, analytics providers, or data brokers involved in the operation of this extension.

Changes to this policy

If we make changes to this privacy policy, we will update this page and the "last updated" date above. Since Forklo collects no data, meaningful changes are unlikely.

Contact

If you have questions about this privacy policy, you can reach us at hello@forklo.com.
